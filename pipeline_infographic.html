The Raw Source

The Election Commission of Thailand (กกต.) publishes the official polling-station results via "Form สส.6/1". However, these are uploaded to Google Drive as flat, scanned PDFs (Canon network scanners) with absolutely zero text layers, completely locked from standard scraping tools.

776 PDF Files No Text Layer Google Drive
02

Multi-Modal AI Extraction (OCR)

Standard PyPDF libraries fail here. To extract the data, a civic-tech pipeline (killernay/election-69-OCR-result) pushes every PDF page through Google Gemini Vision APIs to execute high-fidelity Optical Character Recognition (OCR), translating pixel-data into unstructured text arrays.

Gemini Vision AI Google Cloud Vision Machine Vision
03

Cross-Validation & Data Cleaning

LLMs hallucinate. The pipeline identified 616 wrong candidate numbers and 3,184 "phantom" political parties invented by the OCR. To fix this, algorithms cross-referenced the unstructured text against a verified national database (Reporter DB), repairing misread Thai numerals (e.g., ๖ to ๕) and guaranteeing that all District Vote Sums exactly match Total Ballots.

UUID Lookups Math Verification JSON Structuring
04

Database Assembly

The validated JSON arrays are transformed into flat CSV files. A local Python script (build_database.py) then consumes these files, casting them into a highly structured, relational SQLite database (election_2569.db). Ready for BI tools like Tableau, Pandas, or plain SQL.
